## Running the Web Application

### Production Mode

```sh
cd web
yarn
yarn build
cd ..
fastapi run app.py
```

### Development Mode

1. Start the front-end file watcher:

```sh
cd web
yarn
yarn watch
```

2. Start the back-end server with live reloading:

```sh
fastapi dev app.py
```
